# Support Information

## Getting Support

RDCPartner provides support for the GOTime extension through the following channels:

### Email Support
**Primary Contact**: support@rdcpartner.com

For technical support, license inquiries, or general questions, please email us. We typically respond within 1-2 business days.

### Support Hours
- **Monday - Friday**: 9:00 AM - 5:00 PM (UTC+3)
- **Weekends and Holidays**: Limited availability

## Types of Support

### Technical Support
- Installation and setup assistance
- Troubleshooting technical issues
- Feature questions and usage guidance
- Bug reports

### License Support
- License activation and validation issues
- License renewal inquiries
- Demo license requests
- Billing and payment questions

### Feature Requests
We welcome feature requests and feedback. Please send detailed descriptions to support@rdcpartner.com.

## Before Contacting Support

To help us assist you more effectively, please provide:
- Your Azure DevOps organization/host name
- Extension version
- Description of the issue or question
- Steps to reproduce (if applicable)
- Screenshots or error messages (if applicable)

## Documentation

- **User Guide**: Available in the extension documentation
- **Technical Documentation**: Contact support for access
- **FAQ**: Check extension overview page in Azure DevOps Marketplace

## Service Level Agreement (SLA)

### Response Times
- **Critical Issues**: Within 24 hours
- **General Inquiries**: Within 2 business days
- **Feature Requests**: Acknowledged within 1 week

### Support Limitations
- Support is provided for the current and previous major version of the Extension
- Support does not cover customization or integration with third-party tools beyond documented features
